These R scripts will create voting maps for various legislation, the example given is the 
Birds and Bees Act.

For the Assembly version of these maps, you will need the GetAssemblyBillInfo.R source 
file to parse data from Assembly website. You also will need assembly.vote.names.csv which attaches 
votes in vote listing back to Assembly District number.

For the Senate version of these maps, you will need as Senate API key. Get a key here:
https://legislation.nysenate.gov/static/docs/html/index.html
